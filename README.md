# Positivus

## Descripción

**Positivus** es una *landing page* desarrollada como proyecto personal, orientada a presentar los servicios de una agencia de marketing digital. El objetivo principal del proyecto es practicar y demostrar habilidades en desarrollo web frontend utilizando tecnologías básicas de la web.

La página describe cómo una agencia de marketing digital ayuda a las empresas a crecer y tener éxito en línea mediante servicios como **SEO**, **PPC**, **marketing en redes sociales** y **creación de contenido**.


## Tecnologías utilizadas

- HTML5 – Estructura del contenido  
- CSS3 – Estilos y diseño visual  

## Características

- Diseño tipo landing page
- Estructura semántica en HTML
- Estilos personalizados en CSS
- Proyecto estático
- Enfoque en presentación clara de servicios

## Requisitos previos

No se requieren dependencias ni instalaciones adicionales.  
Solo es necesario un navegador web moderno.
